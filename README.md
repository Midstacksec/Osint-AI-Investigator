# OSINT AI Investigator

Projeto pessoal para estudar OSINT (Open Source Intelligence) usando IA.  
A ideia é ter uma ferramenta simples que ajude a coletar e organizar informações públicas da internet para apoiar investigações e análises.

## Objetivo do projeto

Explorar na prática como juntar técnicas de OSINT com modelos de linguagem.  
Quero algo que facilite testes de cenários reais de investigação, mas de forma didática, focado em aprendizado e experimento.

## Tecnologias utilizadas

- Python no backend
- HTML, CSS e JavaScript no frontend
- Modelos de linguagem (IA) para análise de texto
- Bibliotecas de requisição HTTP para integrar com serviços externos

## Estrutura do projeto

- `backend/`  
  Contém o código do servidor, regras de negócio e o `requirements.txt` com as dependências.

- `frontend/`  
  Interface web da aplicação: `index.html`, `style.css` e `script.js`.
