# OSINT AI Investigator

Ferramenta de apoio a investigações baseada em OSINT (Open Source Intelligence).
O projeto utiliza inteligência artificial para ajudar na coleta e análise de informações disponíveis publicamente na internet.

A ideia é automatizar parte do processo investigativo, reunindo dados de diferentes fontes e apresentando essas informações de forma organizada para análise.

## Tecnologias utilizadas

O projeto foi desenvolvido utilizando:

* Python para o backend
* HTML, CSS e JavaScript na interface web
* APIs de modelos de linguagem para análise de dados
* Bibliotecas de requisição HTTP para integração com serviços externos

## Estrutura do projeto

O repositório está dividido em duas partes principais:

backend/
Contém o código do servidor, lógica da aplicação e o arquivo `requirements.txt` com as dependências do projeto.

frontend/
Interface web da aplicação, incluindo os arquivos `index.html`, `style.css` e `script.js`.

## Como executar o projeto

1. Clone o repositório

git clone https://github.com/Midstacksec/Osint-AI-Investigator.git

2. Acesse a pasta do projeto

cd Osint-AI-Investigator

3. Instale as dependências do backend

pip install -r backend/requirements.txt

4. Execute o servidor

python backend/app.py

Após iniciar o servidor, a interface pode ser acessada pelo navegador.

## Objetivo do projeto

Este projeto foi desenvolvido como estudo e demonstração de como ferramentas de OSINT podem ser combinadas com inteligência artificial para auxiliar processos de investigação e análise de dados públicos.
